# Unidos pelo Bem — Projeto Front-End

Este repositório contém o site institucional da ONG **Unidos pelo Bem**.
Páginas:
- index.html
- projetos.html
- cadastro.html

Estrutura:
```
unidos-pelo-bem/
├─ css/style.css
├─ js/script.js
├─ img/
├─ index.html
├─ projetos.html
├─ cadastro.html
```

Como usar:
1. Baixe e extraia este ZIP.
2. Abra `index.html` no seu navegador.
